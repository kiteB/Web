case 4

Regular Expression: r.+?

Regular Expression: r.?

=> case 1: 'r + any character(0회 이상)'인 형태를 찾아라.
case 2: 'r + any character(* 기호의 최솟값만큼)'인 형태 찾기 => 'r + (any character(0회)'인 형태 찾기 -> 즉 r만 찾기!
case 3: 'r + any character(1회 이상)'인 형태 찾기.
case 4: 'r + any character(+ 기호의 최솟값만큼)'인 형태 찾기. => 'r + (any character(1회)'인 형태 찾기
case 5: 'r + any character(0 또는 1회)'인 형태 찾기.
case 6: 'r + any character(? 기호의 최솟값만큼)'인 형태 찾기. => 'r + (any character(0회)'인 형태 찾기 -> 즉 r만 찾기!

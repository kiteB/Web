?의 예시

case 1

Regular Expression: -X?XX?X

--XX-@-XX-@@-XX-@@@-XX-@@@@-XX-@@-@@-


case 2

Regular Expression: -@?@?@?-

--XX-@-XX-@@-XX-@@@-XX-@@@@-XX-@@-@@-


case 3

Regular Expression: [^@]@?@

--XX-@-XX-@@-XX-@@@-XX-@@@@-XX-@@-@@-

=> case 1: '-' + 'X가 0 또는 1 + X + X가 0 또는 1 + X'인 형태를 찾아라.
즉, -_X_X인 형태를 찾아라. (_자리에는 X가 0 또는 1회 들어갈 수 있음.)

case 2: '-' + '@가 0 또는 1회 + @가 0 또는 1회 + @가 0 또는 1회' + '-'인 형태를 찾아라.
즉, -___-인 형태를 찾아라. (_자리에는 @가 0 또는 1회 들어갈 수 있음.)

case 3: '@가 아닌 어떤 문자 + @가 0 또는 1회 + @'인 형태를 찾아라.
즉, '(any character)_@인 형태를 찾아라. (_자리에는 @가 0또는 1회 들어갈 수 있음.)

수량한정자 '*', '+', '?'는 괄호 표기법의 특별한 케이스. (수량한정자는 대괄호를 이용해서 나타낼 수 있음)
* == {0, }
+ == {1, }
? == {0, 1}

case 1

Regular Expression: AB*A

AA ABA ABBA ABBBA


case 2

Regular Expression: AB{0,}A

AA ABA ABBA ABBBA


case 3

Regular Expression: AB+A

AA ABA ABBA ABBBA


case 4

Regular Expression: AB{1,}A

AA ABA ABBA ABBBA


case 5

Regular Expression: AB?A

AA ABA ABBA ABBBA

=> case 1: 'A + B가 0회 이상 + A'인 형태를 찾아라.
case 2: 'A + B가 0회 이상 + A'인 형태를 찾아라. (Case 1과 완벽하게 동일).
case 3: 'A + B가 1회 이상 + A'인 형태를 찾아라
case 4: 'A + B가 1회 이상 + A'인 형태를 찾아라. (Case 3과 완벽하게 동일)
case 5: 'A + B가 0회 또는 1회 + A'인 형태를 찾아라.

\w (alphanumeric) = alphabet + number + under bar("_")
특수문자, 화이트 스페이스 등은 제외
문자 클래스 [A-z0-9]로도 표현 가능

case 1

Regular Expression: \w

A1 B2 c3 d_4 e:5 ffGG77--__—


case 2

Regular Expression: \w*

A1 B2 c3 d_4 e:5 ffGG77--__—


case 3

Regular Expression: [a-z]\w*

A1 B2 c3 d_4 e:5 ffGG77--__—


case 4

Regular Expression: \w{5}

A1 B2 c3 d_4 e:5 ffGG77--__—


case 5

Regular Expression: [A-z0-9_]

A1 B2 c3 d_4 e:5 ffGG77--__—

=> case 1: 알파벳과 숫자, 언더바만 찾기!
case 2: 알파벳, 숫자, 언더바 0개 이상 찾기
case 3: 'a~z' + alphanumeric 0개 이상 찾기
case 4: alphanumeric 5개 이상
case 5: 'A~z, 0~9, _'을 찾아라 == alphanumeric을 찾아라!
